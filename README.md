# Assignment Management System

## Overview

Assignment Management System is a mobile application designed in Figma to streamline assignment management for both students and teachers. The application provides role-based access, enabling students to track assignments, monitor deadlines, submit work, and view marks, while teachers can create assignments, manage submissions, and assign marks through dedicated dashboards.

---

## Problem Statement

Students often face difficulties in tracking assignment deadlines, monitoring pending work, and viewing marks in a centralized platform. Teachers require an efficient system to create assignments, review submissions, and manage grading activities.

---

## Solution

The Assignment Management System provides a structured and user-friendly platform that simplifies academic workflows. The application offers separate interfaces for students and teachers, improving assignment tracking, submission management, and grading processes.
